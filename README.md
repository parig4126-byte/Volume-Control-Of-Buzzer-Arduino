# Volume-Control-Of-Buzzer-Arduino
An Volume controller of buzzer using potentiometer and arduino 

# Component Used 
1. Arduino R4 WIFI
2. Potentiometer
3. Buzzer
4. Jumper wires
5. Breadboard

# Connections 
1. Encoder Connections:
~ ​GND -> Arduino GND
​~ + (VCC) -> Arduino 5V
~ ​CLK -> Arduino Pin 2 
~ ​DT -> Arduino Pin 3 
2. ​Buzzer Connections:
~ ​GND (-) -> Arduino GND
~ ​VCC (+) -> Arduino 5V
~ ​I/O (S) -> Arduino Pin 9

# How to Run
1. Open the `sketch_mar27buzzer.ino` file in the Arduino IDE.
2. Connect your Arduino to your computer.
3. Select your Board and Port from the Tools menu.
4. Click **Upload**.
